## Luban 数据表（初次定义）

以下为装扮系统所需的基础配置表。字段名大驼峰（PascalCase），符合 Luban → C# 的导出规则。

### 枚举定义

```csharp
enum EItemType
{
    Dog,          // 小狗形象
    Hat,          // 帽子
    Eyewear,      // 眼镜
    Arm,          // 手臂衣服
    Wrist,        // 手腕饰品
    Table,        // 牌桌
    Background,   // 背景
    Accessory,    // 嗜好物品
}

enum ERarity
{
    Common,       // 普通
    Rare,         // 稀有
    Epic,         // 史诗
    Legendary,    // 传说
    Special1,     // 特殊1
    Special2,     // 特殊2
}
```

### 物品配置表 `TbItem`

| 字段 | 类型 | 说明 |
|------|------|------|
| Id | int | 唯一 ID |
| Name | string | 物品名 |
| ItemType | EItemType | 物品类型 |
| Rarity | ERarity | 稀有度 |
| AssetPath | string | 资源路径，如 `Assets/Eyewear/EyePatch.png` |
| SortOrder | int | 排序权重，数字越大约靠前 |

### 盲盒规则表 `TbBlindBox`（预留）

后续扩展。

### 玩家存档数据结构（非 Luban）

运行时数据，通过 `user://save.dat`（ConfigFile / JSON）读写。

```csharp
struct InventoryEntry
{
    int ItemId;     // 关联 TbItem.Id
    int Count;      // 持有数量
    bool IsNew;     // 是否带 New 标记
}
```

存档文件示例：

```ini
[inventory]
item_1=1,true      ; ItemId=1, Count=1, IsNew=true
item_5=3,false     ; ItemId=5, Count=3, IsNew=false

[blind_box]
claim_count=3
current_cost=10000
```
