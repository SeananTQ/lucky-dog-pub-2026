# 消耗品与幸运发牌系统说明

## 概述

消耗品 是玩家在牌桌上使用的消耗品，例如酒水、茶水或小食。

当前阶段只实现 消耗品 的显示、确认使用、扣除数量和占位反馈。幸运发牌的实际算法、数值效果和数据驱动规则留到后续阶段处理。

## 摆放规则

玩家可以在背包中点击 消耗品 类物品，将其摆放到牌桌上的 消耗品 位置。

摆放行为只改变牌桌上的显示物品，不会立刻扣除背包数量，也不会立刻产生 Buff。

如果玩家在背包中选择另一个 消耗品，牌桌上的显示物品会切换为新选择的 消耗品。

## 使用确认

玩家点击牌桌上的 消耗品 后，系统弹出气球提示。

气球提示提供使用入口：

1. 玩家点击“Use”后，系统扣除 1 个当前 消耗品，并打印占位日志，例如“消耗 XXX”。
2. 玩家点击“X”或点击气球外区域后，气球关闭，本次使用流程取消。

气球提示不是弹窗，不应阻断牌桌操作。

## 占位 Buff

当前阶段使用占位 Buff 表示 消耗品 已经被消耗。

占位 Buff 不改变发牌概率，只记录一个后续可接入幸运发牌系统的状态。

Buff 按牌局次数计算，而不是按真实时间计算。这样玩家切换到桌宠模式后，不会因为离开牌桌而浪费 Buff。

第一版可以使用固定剩余次数。后续接入数据驱动时，再由 消耗品 配置决定持续局数和幸运值强度。

## 生效期间交互

Buff 生效期间，玩家仍然可以在背包中更换牌桌上显示的 消耗品。

更换牌桌显示物不会影响已经生效的 Buff。生效中的 Buff 仍然来自之前被消耗的 消耗品。

Buff 生效期间，玩家点击或悬停牌桌上的 消耗品 时，系统显示剩余生效次数的气球提示。每次点击都应重新播放气球提示弹出的动画。

## Buff 结束

Buff 剩余次数归零后，系统根据牌桌当前显示物处理：

1. 如果牌桌上的 消耗品 与最后一次产生 Buff 的 消耗品 相同，则牌桌上的 消耗品 取消显示。
2. 如果牌桌上的 消耗品 已被玩家更换为其他种类，则保留当前显示物，并弹出气球提示说明上一段效果已经结束。

玩家关闭该提示后，再次点击牌桌上的 消耗品，会重新进入“Use / X”的使用确认流程。

## 后续扩展

后续幸运发牌系统需要单独设计以下内容：

1. 消耗品 对应的幸运值强度。
2. Buff 持续局数。
3. Buff 是否允许叠加或替换。
4. 幸运值如何影响发牌算法。
5. 发牌算法如何从硬编码迁移为数据驱动。
