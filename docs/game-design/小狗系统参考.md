# 小狗系统参考

## 提示信号

狗根据补牌后的实际结果给提示（不是预设等级）。4 档信号：

| 信号 | 视觉 | 对话 |
|------|------|------|
| `Bored` | 打哈欠 | "*yawns* Not looking great..." |
| `Happy` | 摇尾巴 | "*wags tail* Feeling good!" |
| `LuckyEye` | 眼睛发光 | "*eyes light up* Something big!" |
| `TopTier` | 竖起耳朵 | "*ears perk up* INCREDIBLE!" |

## 流程

1. 发牌后玩家可点击小狗 → 狗评估当前保留策略并给出信号
2. 玩家改保留牌后 → 狗自动戴墨镜锁定（不再给提示）
3. 戴墨镜后再点狗 → 摇手拒绝

## 爪子系统

- 默认：双手手背
- `ShowClawPalm()` — 切换到手心
- `ShowClawBack()` — 切换到 手背
- 爪子使用 `z_as_relative = false` 控制叠放层级

## 默认表情

`Eyes_Cute` + `Ears_Happy` + 双手手背
