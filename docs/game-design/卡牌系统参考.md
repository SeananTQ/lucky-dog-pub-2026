# 卡牌系统参考

## 卡牌编码

- 0-51 整数编码：`suit = card / 13` (0=Club,1=Diamond,2=Heart,3=Spade)，`rank = card % 13` (0=Ace,...,12=King)
- 文件名：`CardToString()` → `Club1` ~ `Spade13`（rank+1，美术资源从 1 开始）

## 保留/弃牌交互

- 默认全保留（正面朝上）
- 点击卡牌 → 翻到背面（弃牌）
- 再点击 → 反悔翻回正面
- 补牌时，背面的牌翻转变成新牌

## 卡牌动画

| 动画 | 方法 | 说明 |
|------|------|------|
| 发牌 | `AnimateDeal(delay)` | 从高处滑入 + 阴影变化 + 翻转 |
| 弃牌 | `AnimateDiscard()` | 正面→背面翻转（与阴影同步） |
| 反悔 | `AnimateRehold()` | 背面→正面翻转（与阴影同步） |
| 补牌 | `AnimateReplace()` | 背面→新牌面翻转（与阴影同步，逐个播放） |
