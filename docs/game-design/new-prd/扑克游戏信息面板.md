# 扑克游戏信息面板

## 概述

游玩模式下，信息面板在扑克游戏面板左侧（或右侧，超出屏幕时切换），尺寸 240×600。

该面板仅存在于游玩模式。沉浸模式不显示信息面板（信息集成在 HUD 内）。

## 布局

@AI [奖励表从高到低，最上面是皇家同花顺，底部是玩家取得的]
```
┌──────────────────┐
│ 💰 :     88,888  │  ← 玩家持有筹码
├──────────────────┤
│ Jacks or Better  │  ← 当前段位
│ Jacks or Better  │     5
│ Two Pair         │    10
│ Three of a Kind  │    20
│ Straight         │    30
│ Flush            │    40
│ Full House       │    50
│ Four of a Kind   │    60
│ Straight Flush   │    80
│ Royal Flush      │   100
│                  │  ↑ 牌型+奖励对照表
├──────────────────┤
│ 🎁  Bling Box    │
│     10000        │  ← 盲盒入口（预留）
├──────────────────┤
│ ⚙ Settings       │  ← 系统功能按钮
└──────────────────┘
```

## 功能

### 筹码与段位
- 显示当前筹码数，格式（`88,888`）
- 显示当前段位名称（如 "Jacks or Better"）

### 牌型奖励对照表
- 九种牌型 + 对应的奖励倍率（当前下注 × 倍率）
- 高亮玩家最近一次结算时的牌型行

### 盲盒入口（预留）
- 按钮 + 消耗筹码数文字
- 当前版本：按钮不可点击，文字固定显示 `88888`
- 后续版本由盲盒系统驱动

### 系统功能按钮
- 点击后弹出系统功能面板（覆盖层形式）
- 默认定位在扑克游戏面板上方，复用 PanelPositioner 智能定位
