# 游戏面板

@AI [游戏面板在玩家视角下有3种模式，但我们具体实现成3个面板，还是1个面板的3种形态带讨论。]

## 桌宠伪装模式

又称老板来了模式，即将自身伪装成Codex那种普通的桌宠，玩家可通过设置中的快捷键将游戏快速置为老板来了模式。
实际视觉表现应该就是一只小狗，小狗拔在一个矩形方框上，方框内有文字显示，方框右侧有个齿轮点击之后可以显示系统功能面板。
在设置中可以调节文字显示的两种模式：
 - 时钟模式 显示当前的时间
 - 计数模式 显示玩家持有的筹码数量，超过方框能显示的位数后，则改成`88.8K`这种显示形式

该模式下小狗包含其穿戴的装扮，目前版本具体就是帽子、眼镜。
该模式下不显示玩家的手臂、牌桌、背景等。

该模式下是否显示对话气泡，可在设置中更改。

## 桌宠游玩模式

该模式为标准游玩模式。
该模式由两个面板组成，画面尺寸大致为840x600，
默认情况下600x600是我们之前做好的扑克游戏面板，其左侧是240x600的扑克游戏信息面板，如果左侧超出了屏幕，则扑克游戏信息面板则会显示在扑克游戏面板的右侧，其内部具体功能详见[扑克游戏信息面板](./扑克游戏信息面板.md)

## 沉浸游玩模式

窗口话全屏模式的一种游玩模式，采用基于1200x1200尺寸的美术素材。

除了桌宠游玩模式的功能之外，
该模式下会播放游戏背景音乐、音效等完整体验。

玩家可通过老板来了键快速切换至桌宠伪装模式。

#### 装扮部位页签 小狗
##### 小狗的形象图标背包

小狗的形象图标网格，类似网游中背包网格那种感觉，玩家可以点击其中一个图标来“穿戴”它。
新获得的小狗形象图标上带有`New`标记。

等游戏数据表建立后，会按照稀有度从高到低排列，现阶段按照资源名称字母顺序排列a-z。
带有New标记的图标优先排列在最前面，其内部再按照排序规则排列一次。

玩家点击一次图标进行穿戴交互即可移除New标记。

小狗的形象图，需美术或工具单独生成资源，即图片内包含狗头、耳朵、舌头、爪子、和一个模拟windows任务栏的颜色条。将他们合成一张图片，并缩小画布尺寸到256或者512形成美术资产，在本功能内继续缩小该美术资产使其编程图标。

## 广告位
该功能其实是重点，因为本项目的定位是一个非盈利的游戏，重点是增加影响力，为后续大型游戏积累粉丝用户。
所以该功能应该有点类似LinkTree的功能。

平时会有一些链接Banner，可能需要联网更新，或者是纯文本的链接标题，玩家点击之后就会打开相应的网页页面，同时为玩家发放相应奖励。
不要求页面callback，即不要求网页页面真的识别到了玩家打开查看的具体信息然后通知游戏。
但要想办法记录该玩家领取过该奖励，避免重复领取。
奖励就是个盲盒。
