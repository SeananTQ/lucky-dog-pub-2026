# 游戏模块与面板

## 术语约定

| 术语 | 说明 |
|------|------|
| **宿主窗口** | 唯一的 OS 窗口，透明 + 无边框 + 置顶 |
| **面板** | 宿主窗口内可独立显示/隐藏的 UI 控件 |
| **页面** | 系统功能面板内的标签页（设置、调试、装扮等） |
| **覆盖层** | 半透明遮罩 + 居中面板，用于系统功能、游戏结束等模态场景 |

## 面板层级

以下为从顶到底的 z-layer 结构：

```
宿主窗口（OS 窗口）
├── 伪装模式面板（CanvasLayer, Boss Key 模式独占）
│   ├── 狗 Sprite
│   ├── 文字 Label（时钟/筹码）
│   ├── 系统按钮 ⚙
│   ├── 模式切换按钮 🔄
│   └── 信息气泡
├── 游玩/沉浸模式内容
│   ├── 游戏面板（Node2D, 扑克内容）
│   │   ├── 背景、牌桌、卡牌、小狗、手臂等
│   │   └── HUD（CanvasLayer, 消息/按钮/设置入口）
│   ├── 信息面板（CanvasLayer, 240×600, 仅游玩模式）
│   │   ├── 筹码 + 段位
│   │   ├── 牌型奖励表
│   │   ├── 盲盒入口（预留）
│   │   └── 系统按钮
│   └── 系统功能面板（CanvasLayer, 覆盖层）
│       ├── 半透明遮罩
│       └── 功能页面（标签页切换）
│           ├── 游戏设置
│           ├── 调试设置
│           ├── 装扮管理
│           ├── 图鉴
│           └── 广告位
└── 游戏结束覆盖层（CanvasLayer, 现有 Overlay）
```

## 系统功能面板页面说明

### 游戏设置

UI 行为：所有折叠区域头点击后展开/收起，互不影响（不像手风琴）。

#### 必要设置（默认展开）
| 控件 | 类型 | 说明 |
|------|------|------|
| 系统提示 | TextArea | 游戏内全局消息/公告 |
| 多语言 | Dropdown | 中文 / English / 等 |
| 肤色 | Dropdown | Emoji 🧑🏻🧑🏼🧑🏾🧑🏽, 对应手臂颜色 |
| 多显示器 | Dropdown | 选择游戏显示在哪台显示器上 |
| 开机自启动 | Toggle | 注册到 Windows 启动项 |

#### 显示设置
| 控件 | 类型 | 选项 | 说明 |
|------|------|------|------|
| 游戏面板缩放 | Dropdown | 0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x | 影响该面板内所有内容 |
| 系统面板缩放 | Dropdown | 同上 | 影响系统功能面板内所有内容 |
| ≈ 按钮 | 推荐值 | 根据显示器 DPI 计算 | 放在缩放下拉菜单右侧 |

#### 系统设置
| 控件 | 类型 | 说明 |
|------|------|------|
| 始终置顶 | Toggle | 宿主窗口始终在顶部 |
| 显示任务栏图标 | Toggle | 在 Windows 任务栏显示/隐藏 |

#### 隐私设置
| 控件 | 类型 | 说明 |
|------|------|------|
| Discord Rich Presence | Toggle | 在 Discord 展示游玩状态 |
| Kook 状态 | Toggle | 在 Kook 展示游玩状态 |

#### 交互设置
| 控件 | 类型 | 说明 |
|------|------|------|
| 任务栏吸附 | Toggle | 窗口拖拽靠近屏幕边缘时自动吸附 |
| 无干扰模式 | Toggle | 禁用所有非必要事件消息/弹窗 |

### 调试设置

| 控件 | 类型 | 说明 |
|------|------|------|
| 随机种子 | Label + 复制按钮 | 当前牌局种子，点击复制到剪贴板 |
| 固定种子输入 | TextField | 发牌前输入固定种子 |
| 随机换装 | Button | 随机切换玩家/小狗时装 |
| — | 分割线 |
| 重置存档 | Button | 弹出确认框，清空所有数据 |
| 获得全部物品 | Button | 解锁所有装扮 |
| 作弊倍率 | Slider 1-50 | 每局奖励 ×N 倍 |

### 装扮管理

#### 部位页签系统

所有部位页签共用同一套网格 UI 逻辑，仅数据源不同：

| 部位 | 数据源 | 备注 |
|------|--------|------|
| 🐕 小狗 | 小狗形象表 | 狗头+耳朵+舌头+爪子+颜色条 合成缩略图 |
| 🎩 帽子 | 帽子表 | |
| 👓 眼镜 | 眼镜表 | |
| 💪 手臂 | 衣服表 | |
| 📿 手腕 | 饰品表 | |
| 🃏 牌桌 | 牌桌表 | |
| 🖼️ 背景 | 背景表 | |
| 🎁 嗜好 | 嗜好物品表 | |

#### 网格 UI 规则

- 背包网格展示该部位所有可用的物品图标
- 新获得的物品带 `New` 标记
- 排序：`New 物品优先` > 稀有度降序 > 文件名 a-z
- 点击图标穿戴/卸下，穿戴即移除 `New` 标记
- 每个部位只能穿戴一个物品

### 广告位（LinkTree）

```
╔══════════════════════════════════╗
║  🌐 我的 Steam 主页    [领取]    ║  ← 点击 = 打开网页 + 发盲盒奖励
║  📱 小红书账号         [领取]    ║
║  🐦 微博               [领取]    ║
║  ☕ 赞助我们            [领取]    ║
║  ...                              ║
║  更多链接滚动              ║
╚══════════════════════════════════╝
```

- Banner 或纯文字链接列表
- 点击链接 = `OS.ShellOpen(url)` + 发放盲盒奖励
- 不要求网页 callback 确认
- 本地记录已领取状态，避免重复领取
- 奖励数据: 关联盲盒 ID，后续由盲盒系统定义
