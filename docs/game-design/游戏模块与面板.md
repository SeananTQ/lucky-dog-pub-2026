# 游戏模块与面板

## 术语约定

| 术语 | 说明 |
|------|------|
| **宿主窗口** | 唯一的 OS 窗口，透明 + 无边框 + 置顶。三种模式共享，切换时 resize |
| **A 区** | 宿主窗口内的主内容区域，每种模式尺寸不同，@AI [A区会响应拖拽从而移动宿主窗口的位置。] |
| **面板** | 宿主窗口内可独立显示/隐藏的 UI 控件，在 A 区周围的缓冲区展开 |
| **页面** | 系统功能面板内的标签页（设置、调试、装扮等） |
| **覆盖层** | 半透明遮罩 + 居中提示，用于游戏结束等场景 |

## 三种模式

| 模式 | A 区内容 | A 区尺寸 | 素材缩放 |
|------|---------|---------|---------|
| 伪装模式（BossKey） | 狗 + 文字 + 按钮 | 约 265×235 | 25% |
| 游玩模式 | 扑克游戏 | 600×600 | 50% |
| 沉浸模式 | 扑克游戏（全屏） | 1200×1200 | 100% |

## 面板清单

| 面板 | 尺寸 | 用途 | 可用模式 |
|------|------|------|---------|
| 信息面板（C） | 240×600 | 筹码/段位/牌型表 | 游玩、沉浸 |
| 系统功能面板（B） | 800×600（暂定） | 设置/调试/装扮等，标签页切换 | 全部 |
| 测试设置面板 | 420×420（暂定） | v0.1 过渡用，后续并入系统功能面板 | 全部 |

## 面板层级

以下为宿主窗口内的节点结构：

```
宿主窗口（OS 窗口）
├── A 区（主内容，位置固定）
│   ├── [伪装模式] 狗 Sprite + 文字 Label + 按钮 + 气泡（CanvasLayer）
│   ├── [游玩模式] 600×600 扑克内容（Node2D + CanvasLayer HUD）
│   └── [沉浸模式] 1200×1200 扑克内容（Node2D + CanvasLayer HUD）
├── 缓冲区（A 区周围的预留空间，面板在此展开）
│   ├── 信息面板 C（240×600，游玩/沉浸模式，A 左侧）
│   ├── 系统功能面板 B（800×600，A 上方，遵循避让）
│   └── 测试设置面板（过渡用，A 左侧，遵循避让）
└── 覆盖层
    └── 游戏结束提示
```

## 游玩模式默认布局

```
  _B(800×600)_
  | 信息面板 | 游戏面板
  | C 240×600 | A 600×600
```

- C 固定在 A 左侧，与 A 等高
- B 在 A 和 C 的上方，默认与 A 左对齐
- 所有面板在宿主窗口初始化时预留空间，展开/收起不改变窗口大小
- 面板位置遵循显示器边界避让规则

## 系统功能面板页面说明

系统功能面板为标签页结构，包含以下页面：

### 游戏设置

#### 必要设置（默认展开）
| 控件 | 类型 | 说明 |
|------|------|------|
| 系统提示 | TextArea | 游戏内全局消息/公告 |
| 多语言 | Dropdown | 中文 / English / 等 |
| 肤色 | Dropdown | Emoji 🧑🏻🧑🏼🧑🏾🧑🏽, 对应手臂颜色 |
| 多显示器 | Dropdown | 选择游戏显示在哪台显示器上 |
| 开机自启动 | Toggle | 注册到 Windows 启动项 |

#### 显示设置
| 控件 | 类型 | 选项 | 说明 |
|------|------|------|------|
| 游戏面板缩放 | Dropdown | 0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x | 影响该面板内所有内容 |
| 系统面板缩放 | Dropdown | 同上 | 影响系统功能面板内所有内容 |
| ≈ 按钮 | 推荐值 | 根据显示器 DPI 计算 | 放在缩放下拉菜单右侧 |

#### 系统设置
| 控件 | 类型 | 说明 |
|------|------|------|
| 始终置顶 | Toggle | 宿主窗口始终在顶部 |
| 显示任务栏图标 | Toggle | 在 Windows 任务栏显示/隐藏 |

#### 隐私设置
| 控件 | 类型 | 说明 |
|------|------|------|
| Discord Rich Presence | Toggle | 在 Discord 展示游玩状态 |
| Kook 状态 | Toggle | 在 Kook 展示游玩状态 |

#### 交互设置
| 控件 | 类型 | 说明 |
|------|------|------|
| 任务栏吸附 | Toggle | 窗口拖拽靠近屏幕边缘时自动吸附 |
| 无干扰模式 | Toggle | 禁用所有非必要事件消息/弹窗 |

| 部位 | 数据源 | 备注 |
|------|--------|------|

| 🎩 帽子 | 帽子表 | |
| 👓 眼镜 | 眼镜表 | |
| 💪 手臂 | 衣服表 | |
| 📿 手腕 | 饰品表 | |
| 🃏 牌桌 | 牌桌表 | |
| 🖼️ 背景 | 背景表 | |
| 🎁 嗜好 | 嗜好物品表 | |

#### 网格 UI 规则

- 背包网格展示该部位所有可用的物品图标
- 新获得的物品带 `New` 标记
- 排序：`New 物品优先` > 稀有度降序 > 文件名 a-z
- 点击图标穿戴/卸下，穿戴即移除 `New` 标记
- 每个部位只能穿戴一个物品


- Banner 或纯文字链接列表
- 点击链接 = `OS.ShellOpen(url)` + 发放盲盒奖励
- 不要求网页 callback 确认
- 本地记录已领取状态，避免重复领取
- 奖励数据: 关联盲盒 ID，后续由盲盒系统定义
