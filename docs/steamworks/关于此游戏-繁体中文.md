# 關於此遊戲 - 繁體中文

你也許已經在電腦上養過貓、魚、小蟲，甚至各種奇妙的桌面夥伴。

但這一次，陪在你身邊的，是一隻小狗。

[動圖：小狗從螢幕角落探出頭，趴在工作列旁，像剛被玩家帶回桌面。]

## 把小狗留在桌面上

Lucky Dog Rise 是一款輕鬆溫暖的桌面寵物遊戲。牠不會佔住你的工作區，也不會打斷你正在做的事；牠只是安靜地待在螢幕角落，陪你工作、讀書、休息，或是在你遊玩其他內容時守在一旁。

當你敲擊鍵盤、點擊滑鼠，或只是繼續自己的電腦日常時，小狗也會用牠自己的方式回應你。那些平常的小動作，會一點一點累積成幸運點數，讓普通的桌面時光多一點被陪伴的感覺。

[動圖：玩家打字和點擊時，小狗在螢幕角落做出表情反應，幸運點數輕輕增加。]

## 累了就和牠玩一局牌

想休息一下時，可以切換到輕鬆的打牌模式。選擇要保留哪些牌，抽取新的牌面，在一點判斷、一點運氣，以及你和小狗之間慢慢培養出的默契中，贏得更多幸運點數。

別只盯著手上的牌。偶爾看看小狗的小表情，也許會發現牠的可愛反應沒有那麼單純。

[動圖：打牌模式中，小狗坐在牌桌旁，玩家選擇保留或捨棄手牌，小狗露出不同的小表情。]

## 蒐集造型，打造你們的小窩

使用幸運點數開啟盲盒，蒐集新的造型、道具和外觀。小狗可以變得更可愛，玩家角色和背景也能一起打扮，讓桌面不只是桌面，而是專屬於你們的小小幸運窩。

[動圖：盲盒打開後掉出造型獎勵，接著切到裝扮介面展示小狗、玩家角色和背景變化。]

## 給牠留一個位置吧

你也許已經有了喜歡的桌面夥伴，也許只是想讓工作和休息的空檔多一點陪伴。

不管是哪一種，在你的電腦桌面上，也留一個位置給這隻小狗吧。
