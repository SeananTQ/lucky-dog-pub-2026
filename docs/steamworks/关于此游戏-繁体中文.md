# 關於此遊戲 - 繁體中文

你也許已經在電腦上養過貓、魚、小蟲，甚至各種奇妙的桌面夥伴。

但這一次，陪在你身邊的，是一隻小狗。

《Lucky Dog Rise》是一款溫暖輕鬆的桌面寵物遊戲。你可以把一隻小狗養在電腦桌面上，讓牠待在螢幕一角，陪你工作、偷閒、打遊戲，讓好運默默溜進你的日常。能遇見你，是小狗最大的幸運，所以牠才叫 Lucky Dog 呀。

牠不會打擾你，也不會催你做什麼。牠只是陪在那裡，看著你完成一件又一件小事，偶爾用自己的方式為你加油。當你遊玩其他全螢幕遊戲時，牠也可以暫時讓出螢幕空間，不打擾你的每一場戰鬥。

當你專心工作時，每一次敲擊鍵盤、每一次點擊滑鼠，都會逐漸累積幸運點數。你的日常操作不再只是單純的輸入和點擊，也會成為小狗成長與收穫的一部分。

當你想放鬆一下時，可以切換到打牌模式，和小狗一起體驗輕鬆的撲克牌小遊戲。選擇要保留哪些手牌，抽取新的牌面，在一點判斷、一點運氣，以及你和小狗之間的默契中，收穫更多幸運點數。

使用幸運點數取得新的盲盒，蒐集造型、道具和全新的外觀。讓這隻一直陪著你的桌面小狗變得更可愛、更特別，也更像真正屬於你的夥伴。

「生活忙忙碌碌，小狗縫縫補補」

在你忙碌的電腦桌面上，也留一個位置給這隻小狗吧！
