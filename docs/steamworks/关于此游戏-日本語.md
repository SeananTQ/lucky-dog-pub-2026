# このゲームについて - 日本語

あなたのパソコンには、もう猫や魚や小さな虫、あるいは少し不思議なデスクトップの仲間がいるかもしれません。

けれど今回、そばにいてくれるのは一匹の子犬です。

[動画：子犬が画面の隅から顔を出し、タスクバーのそばに前足をのせて、デスクトップに連れてこられたように見える。]

## デスクトップに子犬を迎えよう

Lucky Dog Rise は、あたたかくて気軽に楽しめるデスクトップペットゲームです。子犬が作業スペースを占領したり、今していることを邪魔したりすることはありません。画面の片隅にそっといて、仕事や勉強、休憩、ほかのゲームをしている時間に寄り添ってくれます。

キーボードを打つ、マウスをクリックする、いつものようにパソコンで過ごす。そんな何気ない行動に、子犬は小さな反応を返してくれます。日常のひとつひとつが少しずつラッキーポイントになり、いつものデスクトップ時間に小さな幸運とぬくもりを添えてくれます。

[動画：プレイヤーがタイピングやクリックをすると、画面の隅の子犬が反応し、ラッキーポイントが少しずつ増えていく。]

## 疲れたら一緒にカードを1局

少し休みたくなったら、気軽なカードモードへ。どのカードを残すかを選び、新しいカードを引き、判断と運、そして子犬とのあいだに少しずつ育つ小さな絆を頼りに、さらにラッキーポイントを手に入れましょう。

手札だけを見つめていると、見逃すものがあるかもしれません。ときどき子犬の小さな表情にも目を向けてみてください。そのしぐさは、ただかわいいだけではないかもしれません。

[動画：カードモードで、子犬がテーブルのそばに座り、プレイヤーがカードを残すか捨てるか選ぶたびに、子犬が小さな表情を見せる。]

## 衣装を集めて、ふたりの居場所を作ろう

ラッキーポイントでブラインドボックスを開け、新しい衣装やアイテム、見た目を集めましょう。子犬をもっとかわいくしたり、プレイヤーキャラクターや背景をカスタマイズしたりして、デスクトップをあなたと子犬だけの小さな幸運の居場所に変えていきます。

[動画：ブラインドボックスが開いて衣装報酬が出現し、その後カスタマイズ画面で子犬、プレイヤーキャラクター、背景の変化を見せる。]

## この子の場所を空けてあげませんか

すでにお気に入りのデスクトップの仲間がいるかもしれません。あるいは、仕事と休憩の合間に、もう少しだけ寄り添ってくれる存在がほしいのかもしれません。

どちらだとしても、デスクトップの片隅なら、もう一匹分の場所くらい残っているはずです。あなたのデスクトップにも、この子犬の居場所を作ってあげてください。
