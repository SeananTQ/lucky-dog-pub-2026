# 关于此游戏 - 简体中文

你也许已经在电脑上养过猫、鱼、小虫，甚至各种奇妙的桌面伙伴。

但这一次，陪在你身边的，是一只小狗。

[动图：小狗从屏幕角落探出头，趴在任务栏边，像刚被玩家带回桌面。]

## 把小狗留在桌面上

Lucky Dog Rise 是一款轻松温暖的桌面宠物游戏。它不会占据你的工作区，也不会打断你正在做的事情；它只是安静地待在屏幕一角，陪你工作、学习、休息，或是在你游玩其他内容时守在一旁。

它不只是重复同一个动作。你的敲键、点击和停顿，都会成为它观察你的小线索。忙起来时，它会跟上你的节奏；安静下来时，它也会用自己的小表情陪在旁边，像是在为你完成的一件件小事加油。

和传统电子宠物不同，你不需要特意停下来照顾它。只要继续使用电脑，那些平常的输入和点击就会一点点积累成幸运点数，推动小狗与你的桌面生活一起成长。

[动图：玩家打字和点击时，小狗在屏幕一角做出表情反馈，幸运点数轻轻增加。]

## 累了就和它玩一局牌

想休息一下时，可以切换到轻松的打牌模式。选择保留哪些牌，抽取新的牌面，在一点判断、一点运气，以及你和小狗之间慢慢培养的默契中，赢取更多幸运点数。

别只盯着手里的牌。偶尔看看小狗的小表情，也许会发现它并不只是随机卖萌。

[动图：打牌模式中，小狗坐在牌桌旁，玩家选择保留或弃掉手牌，小狗露出不同的小表情。]

## 收集装扮，打造你们的小窝

使用幸运点数开启盲盒，收集新的装扮、道具和外观。小狗可以变得更可爱，玩家角色和背景也能一起打扮，让桌面不只是桌面，而是属于你们的小小幸运窝。

[动图：盲盒打开后掉出装扮奖励，随后切到装扮界面展示小狗、玩家角色和背景变化。]

## 给它留一个位置吧

你也许已经有了喜欢的桌面伙伴，也许只是想让工作和休息的间隙多一点陪伴。

如果你的桌面上还有一点空位，就把它留给这只小狗吧。
