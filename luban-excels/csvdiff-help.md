# csvdiff 使用说明

比对 CSV 文件，生成 Markdown 格式的 diff 报表，只显示发生变化的单元格。

## 运行方式

**方式一：双击运行**
```
csvdiff.bat
```

**方式二：命令行**
```bash
python csvdiff.py
```

## 使用模式

### 模式一：比对两个文件

```bash
python csvdiff.py 文件A.csv 文件B.csv
```

直接传入两个 CSV 文件路径，程序自动判断哪个是旧文件、哪个是新文件。

### 模式二：批量比对文件夹

```bash
python csvdiff.py csv文件夹路径
```

拖拽一个文件夹到 bat 上，程序自动扫描文件夹中同基础文件名的 CSV 版本文件并配对。

例如文件夹中有：
- `__enums___Sheet1.csv` + `__enums___Sheet1_1.csv` → 配对
- `卡牌数据表_卡牌总表.csv` + `卡牌数据表_卡牌总表_1.csv` → 配对

## 版本判断规则

- 不带 `_数字` 后缀的为旧版本
- 带 `_数字` 后缀的为新版本（数字越大越新）
- 配对时取版本最小的作为旧文件，版本最大的作为新文件

## 输出

在当前目录生成 `{旧文件名}.diff.md`，内容包括：
- 变更统计
- 每行变更的列、旧值、新值

## 依赖

无需额外依赖，仅使用 Python 标准库。
